# Клієнт (evCharging)

Веб-застосунок на **React**, **TypeScript** і **Vite**: портал користувача, адміністрування мережі станцій і глобальний адмін (тарифи, прогнози тощо). Спілкується з REST API з каталогу `../server`.

## Запуск

```bash
npm install
npm run dev
```

Детальні нотатки з шаблону Vite/ESLint — у [`../docs/client-vite-eslint-notes.md`](../docs/client-vite-eslint-notes.md). Документація проєкту — у [`../docs/`](../docs/).
